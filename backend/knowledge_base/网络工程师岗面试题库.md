# 网络工程师面试题库

---

## 一、网络基础

### OSI 七层与 TCP/IP 五层模型及各层作用？

**OSI 七层**：物理层、数据链路层、网络层、传输层、会话层、表示层、应用层

**TCP/IP 五层**：物理层、数据链路层、网络层、传输层、应用层

各层作用：

- **物理层**：传输比特流，定义电气/物理规范
- **数据链路层**：成帧、MAC 寻址、差错校验、交换机工作
- **网络层**：IP 寻址、路由选择、路由器工作
- **传输层**：端到端通信、TCP/UDP、端口号
- **应用层**：为应用程序提供网络服务（HTTP、FTP、DNS 等）

### TCP 和 UDP 的区别及应用场景？

- **TCP**：面向连接、可靠传输、流量/拥塞控制、开销大；用于 HTTP、HTTPS、FTP、邮件
- **UDP**：无连接、不可靠、速度快、开销小；用于视频直播、DNS、游戏、语音

### TCP 三次握手详细过程及原因？

1. 客户端发送 SYN 到服务端，请求连接
2. 服务端回复 SYN+ACK，确认并同意连接
3. 客户端回复 ACK，连接建立

目的：确保双方收发能力正常，避免无效连接占用服务端资源。

### TCP 四次挥手详细过程及原因？

1. 主动方发送 FIN，请求关闭
2. 被动方回复 ACK，确认收到
3. 被动方发送 FIN，准备关闭
4. 主动方回复 ACK，连接关闭

原因：双方需独立关闭发送通道，数据传输可能不对称。

### SYN 洪水攻击原理及防御？

攻击者发送大量 SYN 不回复 ACK，占满半连接队列。

防御：SYN-Cookie、限制连接速率、缩短超时、防火墙清洗。

### TCP 可靠传输机制？

序列号、确认应答、超时重传、滑动窗口、流量控制、拥塞控制。

### 滑动窗口、流量控制、拥塞控制？

- **滑动窗口**：控制发送量，提高传输效率
- **流量控制**：接收方告知发送方发送速度，防止缓冲区溢出
- **拥塞控制**：根据网络拥堵情况调整发送速率

### ARP 协议原理及 ARP 欺骗？

ARP 通过 IP 地址获取 MAC 地址。

ARP 欺骗：伪造 ARP 报文篡改网关/主机 MAC，实现监听、断网。

防御：DAI、静态 ARP、端口安全。

### 私有 IP 地址范围？

- 10.0.0.0 ~ 10.255.255.255（10/8）
- 172.16.0.0 ~ 172.31.255.255（172.16/12）
- 192.168.0.0 ~ 192.168.255.255（192.168/16）

### 子网划分、VLSM、CIDR？

- **子网划分**：将大网段拆分为小网段，缩小广播域
- **VLSM**：变长子网掩码，不同网段可使用不同掩码
- **CIDR**：无类别域间路由，使用 `/前缀` 表示网段

### 网关作用？

跨网段通信的出口，没有网关无法访问其他网段。

### DNS 工作流程、递归与迭代区别？

**流程**：客户端 → 本地 DNS → 根 DNS → 顶级域 DNS → 权威 DNS → 返回结果

- **递归**：本地 DNS 替客户端完成全部查询
- **迭代**：客户端逐级查询各个 DNS 服务器

### DHCP 四步报文？

Discover → Offer → Request → ACK

### NAT 作用及类型？

作用：节省公网 IP、隐藏内网拓扑

类型：静态 NAT、动态 NAT、PAT（端口多路复用）

### ICMP 协议作用？

网络探测、错误反馈，ping、tracert 基于 ICMP。

### MTU 与 MSS？

- **MTU**：数据链路层最大传输单元，默认 1500
- **MSS**：TCP 最大分段大小，MSS = MTU - 40

### 广播域与冲突域？

- **冲突域**：同一域内同时发送数据会产生冲突；交换机每个端口一个冲突域
- **广播域**：接收同一广播报文的范围；路由器隔离广播域

### 常用网络排错命令？

ping、tracert、arp、ipconfig、netstat、telnet、ssh、route

### HTTP 与 HTTPS 区别？

- **HTTP**：80 端口，明文传输
- **HTTPS**：443 端口，加密传输、身份认证、防篡改

### HTTPS 加密原理？

非对称加密协商会话密钥，对称加密传输业务数据，证书验证身份。

### 长连接与短连接？

- **短连接**：一次请求/响应后断开
- **长连接**：一次连接可多次请求，HTTP 1.1 默认开启

### 网络丢包排查思路？

检查物理链路、端口状态、双工不匹配、环路、拥塞、防火墙策略，光模块、路由震荡。

### 网络延迟高排查？

检查路由路径、带宽拥塞、设备 CPU、NAT 开销、传输距离、队列调度。

### 路由优先级与度量值？

- **优先级**：不同路由协议之间优选，值越小越优先
- **度量值**：同一协议内选最优路径，值越小越优先

### 常见路由优先级（华为）？

直连 0 → 静态 60 → OSPF 110 → RIP 120

### 跨网段通信完整流程？

主机查路由 → ARP 请求网关 → 网关查路由 → 转发到目标网段 → 回包原路返回

## 二、交换技术

### 交换机工作原理？

学习源 MAC 地址 → 查询 MAC 地址表 → 转发/泛洪

### 交换机、集线器、路由器区别？

- **集线器**：物理层，共享介质，半双工
- **交换机**：数据链路层，隔离冲突域，全双工
- **路由器**：网络层，隔离广播域，跨网段转发

### VLAN 作用？

隔离广播域、提升安全性、简化组网、灵活划分业务

### Access/Trunk/Hybrid 端口区别？

- **Access**：接入终端，仅允许一个 VLAN，发送 Untag 帧
- **Trunk**：中继链路，允许多个 VLAN，发送 Tag 帧
- **Hybrid**：华为特有，可自定义 Tag/Untag

### 802.1Q 协议作用？

为以太网帧添加 VLAN 标签，实现跨交换机 VLAN 透传

### 三层交换机与路由器区别？

三层交换机高速硬件转发，用于内网；路由器支持 NAT、VPN、广域网接口

### STP 协议作用？

消除二层环路，修剪冗余链路形成无环树

### STP 选举：根桥、根端口、指定端口？

- **根桥**：桥 ID 最小（优先级 + MAC）
- **根端口**：非根桥到达根桥最优路径端口
- **指定端口**：每网段唯一负责转发的端口

### STP 端口状态？

Disabled、Blocking、Listening、Learning、Forwarding

### RSTP 对 STP 的改进？

精简端口状态、快速收敛、减少拓扑变更延时

### MSTP 作用？

多实例，不同 VLAN 映射不同生成树，实现负载分担

### 二层环路危害？

广播风暴、MAC 漂移、设备 CPU 占用高、网络瘫痪

### 链路聚合 Eth-Trunk/LACP 作用？

增加带宽、链路冗余、负载均衡

### MAC 地址漂移原因及排查？

原因：环路、双主、错接链路

排查：查看日志、shutdown 端口、检查拓扑

### VLAN 间通信三种方式？

单臂路由、三层交换机、路由器物理口分网段

### DHCP Snooping 作用？

防止非法 DHCP 服务器，绑定 IP+MAC + 端口 + VLAN

### DAI 动态 ARP 检测？

基于 DHCP Snooping 绑定表，防御 ARP 欺骗

### IPSG IP 源防护？

防止 IP 伪造、IP 地址冲突

### 交换机环路排查步骤？

查看 MAC 漂移日志 → shutdown 冗余口 → 检查 Trunk → 检查终端回路 → 开启 STP

### 交换机 QoS 作用？

保障关键业务带宽、优先级调度、流量整形

## 三、路由技术

### 路由器工作原理？

查路由表 → 最长匹配 → 转发到下一跳/出接口

### 静态路由与动态路由区别？

- **静态**：手动配置、无开销、适合小型网络
- **动态**：自动学习、适合大型网络

### 路由表构成？

目的网络、掩码、协议、优先级、度量、出接口、下一跳

### RIP 协议特点？

距离矢量、跳数度量、最大 15 跳、周期更新、适合小型网络

### RIP 防环机制？

水平分割、毒性逆转、触发更新、最大跳数

### OSPF 协议特点？

链路状态、区域化、组播更新、快速收敛、适合大型网络

### OSPF 区域 0 作用？

骨干区域，所有区域必须连接区域 0，负责区域间路由转发

### DR、BDR 选举作用？

减少广播网邻接关系，降低 LSA 泛洪

### OSPF 五种报文？

Hello、DBD、LSR、LSU、LSAck

### OSPF 邻接建立过程？

Down → Init → 2-Way → ExStart → Exchange → Loading → Full

### OSPF 与 RIP 区别？

OSPF 链路状态、无环、快收敛、支持大规模；RIP 距离矢量、有环、慢收敛

### BGP 协议作用？

自治系统间路由协议，控制路由策略

### IBGP 与 EBGP 区别？

EBGP 跨 AS；IBGP 同 AS，需全互联或路由反射器

### BGP 选路原则（前 8）？

优先级 → Local_Pref → 本地始发 → AS-Path 短 → Origin → MED → EBGP＞IBGP → IGP 开销

### VRRP 协议作用？

网关冗余，多台路由器虚拟一个网关

### VRRP 主备选举？

优先级高为主，相同则接口 IP 大为主

### BFD 作用？

快速故障检测，毫秒级感知链路中断

### 标准 ACL 与扩展 ACL？

标准仅过滤源 IP；扩展过滤源/目的 IP、端口、协议

### GRE 隧道作用？

三层 VPN，封装跨网传输

### IPSec VPN 作用？

加密传输，保障公网数据安全

### 路由不通排错步骤？

查直连 → 查网关 → 查路由表 → 查下一跳 → 查 ACL → 查 NAT → 查链路

### OSPF 邻居起不来原因？

区域 ID、认证、掩码、Hello/Dead、静默接口、物理 down

### 路由环路产生原因？

双向引入、过滤不当、静态路由配置错误

## 四、网络安全

### 防火墙三大区域？

Trust、Untrust、DMZ

### 包过滤与状态检测区别？

包过滤仅检查报文头；状态检测跟踪会话，更安全

### ACL 作用？

访问控制、流量过滤、路由过滤、策略匹配

### 常见网络攻击？

ARP 欺骗、DHCP 欺骗、SYN 洪水、DDoS、DNS 劫持、端口扫描、暴力破解

### IPS/IDS 区别？

IDS 检测告警；IPS 在线阻断

### VPN 类型？

IPSec、SSL、L2TP、GRE

### 对称加密与非对称加密？

对称加密快；非对称加密安全，用于密钥协商

### CA 证书作用？

身份认证、防篡改、防伪造

### 零信任安全理念？

默认不信任，持续验证，最小权限

### WAF 作用？

Web 应用防火墙，防御 SQL 注入、XSS

### 堡垒机作用？

统一运维、操作审计、权限管控

### 网闸作用？

物理隔离，内外网安全数据摆渡

### DDoS 防御？

流量清洗、黑洞路由、限流、扩容

## 五、无线 WLAN

### 802.11a/b/g/n/ac/ax 区别？

ax 即 WiFi 6，速度更快、并发更高、延迟更低

### 胖 AP 与瘦 AP 区别？

胖 AP 独立工作；瘦 AP 由 AC 统一管理

### AC+Fit AP 架构？

AC 集中管理，Fit AP 提供接入，CAPWAP 隧道

### 无线漫游原理？

终端在 AP 间无缝切换，保持 IP 不变

### 2.4G 与 5G 区别？

2.4G 穿墙好、干扰大；5G 速度快、干扰小

### 无线加密方式？

WEP 不安全；WPA2/WPA3 企业版安全

### CAPWAP 协议作用？

AC 与 AP 间控制与数据隧道

### 信道规划原则？

2.4G 用 1、6、11；5G 自动信道

### 无线故障排查？

信号、干扰、关联、认证、DHCP、路由

## 六、运维与排错

### 网络不通七层排查法？

物理 → 数据链路 → 网络 → 传输 → 应用 → 安全 → 配置

### 能 ping IP 但打不开网页？

DNS、端口、防火墙、代理、HTTP 服务、MTU

### 内网正常外网不通？

默认路由、NAT、出口链路、ACL、运营商故障

### 环路快速处理？

shutdown 端口 → 查拓扑 → 开启 STP → 查终端回路

### DHCP 获取不到地址？

地址池、网关、路由、Snooping、接口 VLAN、链路

### DNS 解析失败？

DNS 地址、连通性、防火墙、缓存、域名状态

### 丢包严重排查？

链路、双工、拥塞、攻击、环路，光衰

### 防火墙策略不通？

域间策略、源/目的、端口、时间、NAT、会话

### 链路聚合不生效？

模式、成员口、速率双工、对端配置

### 网络割接流程？

方案 → 备份 → 验证 → 回退 → 测试 → 割接 → 验收

### 设备升级注意事项？

版本兼容、备份、空闲时间、回退方案

## 八、综合面试题

### 介绍你做过的网络项目？

按背景 → 架构 → 设备 → 配置 → 优化 → 效果讲述

### 企业网络架构设计？

出口冗余 → 核心冗余 → 汇聚 → 接入 → 无线 → 安全 → 管理

### 如何提高网络可靠性？

冗余，双归、STP/VRRP/BFD、双出口、备份、监控

### 遇到最难的故障？

按现象 → 排查 → 定位 → 解决 → 总结讲述

### 大型网络 IP/VLAN/路由规划？

分段、汇总、区域化、策略、安全隔离

### 网络割接不中断业务？

闲时、备份、分步、回退、验证、监控

### 如何防止内网攻击？

认证、隔离、ACL、DAI、IPSG、端口安全
